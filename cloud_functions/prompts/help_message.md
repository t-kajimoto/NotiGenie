# NotiGenie ヘルプ

NotiGenieは、あなたのNotionデータベースを操作するためのAIアシスタントです。LINEや音声入力から操作できます。

## できること

- **データベース検索**: 「〜のドキュメント探して」「今日のタスク教えて」
- **ページ作成**: 「〜というタスクを追加して」「買い物リストに卵を追加」
- **ページ更新**: 「〜を完了にして」「ステータスを変更して」
- **メモ追記**: 「〜のメモに追記して」

## コマンド

- `help`, `ヘルプ`, `へるぷ`: このメッセージを表示します。

詳しくはNotion上のドキュメントを参照してください。
